# Image Assets

This directory should contain the following images:

1. `parcel-placeholder.png` - A placeholder image for parcels
2. `truck-placeholder.png` - A placeholder image for trucks
3. `logo.png` - The mziGO logo

Please ensure all images are optimized for web use and have appropriate licenses for commercial use.

Recommended image specifications:
- Format: PNG or WebP
- Resolution: At least 2x for retina displays
- Maximum file size: 200KB per image
- Dimensions:
  - Logo: 200x200px
  - Placeholder images: 400x300px
